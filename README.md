# Loan Prediction Analysis (Data Analytics Project)

This project analyzes loan application data to identify the key factors influencing loan approval decisions. Using Python-based data analysis techniques, the project uncovers patterns in applicant income, credit history, education, and property area while preparing the data for predictive analytics.

---

## 📌 Project Overview

The objective of this project is to perform exploratory data analysis (EDA) on loan application records and generate actionable insights from applicant demographics and financial information.

Key business questions addressed:

- What factors most influence loan approval?
- How does credit history affect approval rates?
- What is the relationship between applicant income and loan amount?
- Which applicant groups have higher approval probabilities?

## 📊 Analysis Performed

### Data Cleaning
- Handled missing values using mode and median imputation
- Identified and treated outliers using the IQR method

### Exploratory Data Analysis
- Loan approval distribution analysis
- Credit history impact assessment
- Income and loan amount analysis
- Property area and education comparisons
- Correlation and trend analysis

### Data Preparation
- Feature encoding using Label Encoding
- Feature scaling using StandardScaler
- Dataset preparation for predictive modeling

---

## 💡 Key Insights

- Credit History is the strongest indicator of loan approval.
- Applicants with positive credit history have significantly higher approval rates.
- Graduate applicants show better approval outcomes than non-graduates.
- Applicant income and loan amount are positively correlated.
- Urban and Semi-Urban areas account for a larger share of approved loans.
